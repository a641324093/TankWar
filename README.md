# 坦克大战 TankWar
> 该项目是大二时候初学Java时做的，自己制作的一个简单的游戏用于入门，对于熟悉游戏的逻辑有一定的帮助。

## 游戏截图

![这里写图片描述](https://github.com/a641324093/TankWar/raw/master/mdImg/tankwar.png)
## 开发平台
Java + AWT控件 
Eclipse IDE
## 游戏内容
双方可操作个一台坦克，且有多个AI坦克，有爆炸特效，医疗包以及不可摧毁的墙壁。游戏中的物体均使用图片实现。
